  • The INI file format is an informal standard for configuration files for some platforms of software. Ini files are simple text files with a basic structure composed of "sections" and "properties"
  • The named "INI files" comes from the commonly used filename extension, .INI, which stands for "initialization". Other common initialization file extensions are .CFG, .cong and .txt.
